TIPOGRAFIAS

Las propiedades para cambiar el estilo de la fuente son:

  • font-family
  • font-size: Establece el tamaño de la fuente
    •     PX: pixeles (medida absoluta)
    em: medida relativa al contexto (al padre)
    rem: medida relativa al hdml
    %: medida relativa al tamaño actual

    vh: medida realativa al height del viewport
    vw: medida relativa al width del viewport
    vmin: medida relativa al valor minimo de viewport
    vmax: medida relativa al valor maximo de viemport
  • font-weight: Establece el grosor de la fuente
  • font-style: Establece el estilo de la tipografia

  • text-transform: Transforma el texto a mayuscula o minuscula, etc
  • text-align
  • text-decoration
  • text-indent

  • line-height: centra el texto vertical
  • letter-spacing: Establece la separacion de las letras

Tablas

  • table-layout:

    • Define como se comportan las dimensiones de una tabla y los anchos de las columnas

  • caption-sede:

    • Define donde se coloca el caption de una tabla.

    top: valor por defecto
    bottom: se coloca al pie de la tabala

  • border-spacing:

    • Controla el espacio entre las celdas recibe una medida en cualquier unidad

  • border-collapse:

    • Controla se las celdas se mantienen juntas o separadas

  • empty-cells

    • controla que hacer con las celdas vacias

    show: valor por defecto
    hide: ocultar las celdas vacias
